# Linux Filesystem Hierarchy (FHS)

Understand the main directories in Linux:

- `/boot` Ôćĺ boot files  
- `/cdrom` Ôćĺ CD-ROM media  
- `/dev` Ôćĺ hardware devices  
- `/etc` Ôćĺ system configuration  
- `/home` Ôćĺ user directories  
- `/media` Ôćĺ removable media  
- `/mnt` Ôćĺ temporary mounts  
- `/opt` Ôćĺ optional software  
- `/proc` Ôćĺ process information  
- `/root` Ôćĺ root home  
- `/run` Ôćĺ runtime data  
- `/srv` Ôćĺ service data  
- `/tmp` Ôćĺ temporary files  
- `/usr` Ôćĺ user system files and binaries  
- `/var` Ôćĺ logs, caches, variable data
